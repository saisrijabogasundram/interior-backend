🏠 Interior Design Platform
Transform Your Space Into A Dream Home
A full-stack interior design platform connecting homeowners with expert interior designers. Browse stunning designs, book consultations, shop premium products, and manage your dream home project — all in one place.

✨ Features
For Customers

Browse interior designs filtered by room type, style, and budget
View and book expert interior designers for a free site visit
Shop premium furniture, lighting, and decor products
Add products to cart and place orders
Track projects and manage design journey
Cost estimation tool for budget planning
Personal dashboard with bookings and project overview

For Admin / Owner

Full analytics dashboard with user and booking stats
Manage all bookings and update statuses
Manage staff, designers, and products
View detailed reports and insights

For Staff

Manage and update booking statuses
Add, edit, and remove products
View and manage designer profiles


🛠️ Tech Stack
Frontend — React.js, React Router DOM, Axios, React Icons, CSS Modules
Backend — Django 6.0, Django REST Framework, Simple JWT, CORS Headers, Gunicorn, Whitenoise
Database — MySQL hosted on Railway
Deployment — Frontend, Backend, and Database all hosted on Railway
